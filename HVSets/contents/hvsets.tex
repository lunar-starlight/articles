\section{Heyting-valued Sets}

\index{\subsection}\subsection{Preliminaries}

\begin{definition}
  A \emph{Heyting-valued set \(A\)} is a carrier set \(\carr A\) equipped with a ``Heyting-valued relation'' \(\eq[A]{-}{-} : A├ŚA Ôćĺ \h\) satisfying
  \begin{gather}
    \label{hvs:symm}  \eq x y ÔëĄ \eq y x\\
    \label{hvs:trans} \eq x y Ôłž \eq y z ÔëĄ \eq x z
  \end{gather}

  Define \(\e x Ôëö \eq x x\).
\end{definition}
The idea is, that we wish to systematically handle \emph{presentations} of sheaves.
We think of elements of \(\carr A\) as the ``generators'' of a sheaf and the relation \(\eq[A]{}{}\) to represent the set of equations under which the generators should be quotiented.
Then \(\e x\) represents the \emph{extent of definition} of a generator.

\begin{definition}
  A \emph{Heyting-valued morphism \(f : A ÔćČ B\)} is a ``Heyting-valued functional relation'', meaning it is a map \(\carr f : \carr A ├Ś \carr B Ôćĺ \h\) satisfying
  \begin{gather}
    \label{hvm:str}    f(x,y) ÔëĄ \e x Ôłž \e y\\
    \label{hvm:well}   \eq {x'} x Ôłž f(x, y) Ôłž \eq y {y'} ÔëĄ f(x', y')\\
    \label{hvm:unique} f(x, y) Ôłž f(x, y') ÔëĄ \eq y {y'}\\
    \label{hvm:total}  \e x ÔëĄ Ôőü_{y Ôłł B}f(x, y)
  \end{gather}
\end{definition}

It is generally known that this constitutes a category, that is equivalent to the sheaf topos over the locale \(\h\).

\begin{theorem}[Higgs]
  The category of Heyting-valued sets valued in \(\h\) is equivalent to the category of sheaves on \(\h\).
\end{theorem}

Let us first list some convenient lemmata.
\begin{lemma}
  The following hold.

  \begin{enumerate}
  \item \(f(x,y)Ôłž\eq x {x'} = f(x',y)Ôłž\eq x {x'}\)
  \item \(\e x = Ôőü_{yÔłłA}\eq x y\)
  \item \(\e x = Ôőü_{yÔłłB}f(x,y)\)
  \item \(f(x, y) = Ôőü_{y'ÔłłB}f(x,y')Ôłž\eq y {y'} = Ôőü_{x'ÔłłA}f(x',y)Ôłž\eq x {x'}\)
  \end{enumerate}
\end{lemma}

\begin{construction}
  The product \(A├ŚB\) is given by the carrier \(\carr A ├Ś \carr B\) with equality
  \[ \eq{\p{x,y}}{\p{x',y'}} Ôëö \eq x {x'} Ôłž \eq y {y'}\text. \]

  The projection \(¤Ç : A├ŚB ÔćČ A\) is defined by
  \[ \p{\p{x,y}, x'} ÔćŽ \eq x {x'} Ôłž \e y\text. \]
  The other projection is defined similarly
\end{construction}
\begin{proof}
  Symmetry and transitivity of the equality follow clearly.

  Likewise, the projection is clearly strict and well-defined.
  Uniqueness holds via
  \begin{align*}
    ¤Ç(\p{x,y},x') Ôłž ¤Ç(\p{x,y},x'')
    &= \eq x {x'} Ôłž \e y Ôłž \eq x {x''} Ôłž \e y\\
    &= \eq x {x'} Ôłž \eq x {x''} Ôłž \e y\\
    &ÔëĄ \eq {x'} {x''} Ôłž \e y\\
    &ÔëĄ \eq {x'} {x''}\text.
  \end{align*}
  Totality holds via
  \begin{align*}
    \e{\p{x,y}} = \e x Ôłž \e y = Ôőü_{x'ÔłłA}\eq x {x'} Ôłž \e y\text.\qedhere
  \end{align*}
\end{proof}

\begin{construction}
  Given a Heyting-valued set \(A\) and a relation \(R : \carr A ├Ś \carr A Ôćĺ \h\) satisfying
  \begin{gather}
    \e x ÔëĄ R x x\\

  we can form the \emph{quotient \(\quot A R\)} with carrier \(\carr A\) and equality \(R\).
\end{construction}
\begin{note}
  Subquotients can be formed similarly, by merely dropping the first condition on the relation.
  If instead \(R x x ÔëĄ \e x\) holds the above construction constructs the subset given by the predicate \(R x x\).
\end{note}

\begin{example}
  We can work out an illustrative example.
  Consider the principle \(\lem*\), which says \(\for{p:╬ę}{p ÔłĘ ┬Čp}\).
  Analysing this we get
  \begin{align*}
    \i{\for{p:╬ę}{p ÔłĘ ┬Čp}}
    &= ÔőÇ_{pÔłł\h} {\e p Ôçĺ \i{p ÔłĘ ┬Čp}}\\
    &= ÔőÇ_{pÔłł\h} {\p{p Ôçö p} Ôçĺ p ÔłĘ \p{p Ôçĺ ÔŐą}}\\
    &= ÔőÇ_{pÔłł\h} {ÔŐĄ Ôçĺ p ÔłĘ p^ÔŐą}\\
    &= ÔőÇ_{pÔłł\h} {p ÔłĘ p^ÔŐą}
  \end{align*}

  The principle will hold in \(\h\) precisely when this interpretation equals \(ÔŐĄ\).
  Since it is a conjunction that will hold only when all of its conjuncts (\(p ÔłĘ p^ÔŐą\)) are \(ÔŐĄ\).
  In other words, \(\lem*\) holds over \(\h\) when every element of the Heyting algebra is complemented.
\end{example}
\begin{note}
  The final term in the above computation could be rewritten as \(ÔőÇ_{pÔłł\h\text{ dense}} p\), i.~e. ``the meet of all dense sublocales''.
  This often simplifies computations, for example, sheaves over the Sierpinski space satisfy \(┬Č┬Č\lem*\) but not \(\lem*\), since \(\lem*\) since the open point is the only non-trivial dense open.
\end{note}

\begin{theorem}
  \(\lem*\) holds over \(\h\) iff \(\h\) is a Boolean algebra.
\end{theorem}

We can show a weaker version the same way.

\begin{exercise}
  Show that \(\wlem*\) holds over \(\h\) iff in \(\h\) every pseudo-complement is complemented.

  In the case where \(\h\) is a topology this corresponds to extremally disconnected spaces (closure of every open set is open).
\end{exercise}
\begin{solution}
  Again we compute
  \begin{align*}
    \i{\for{p:╬ę}{┬Čp ÔłĘ ┬Č┬Čp}}
    &= ÔőÇ_{pÔłł\h} {\e p Ôçĺ \i{┬Čp ÔłĘ ┬Č┬Čp}}\\
    &= ÔőÇ_{pÔłł\h} {p^ÔŐą ÔłĘ p^{ÔŐąÔŐą}}
  \end{align*}
  This then by the same argument means that for every \(p Ôłł \h\) we have \(p^ÔŐą\) is complemented.

  In the case of \(\h = ­Łĺ¬X\) saying that an open set is complemented is equivalent to saying it is closed.
  Also, the pseudo-complement in topological spaces is the exterior operation (interior of the complement), so this says that every exterior of an open set is closed.
  Then consider what this means for complements.
  The complement of an exterior is the closure, so then this says ``the closure of every open set is open'', which is exactly the definition of an extremally disconnected space.
\end{solution}

\subsubsection{Intermezzo: Real numbers}

Let \(T\) be some topological space (or in general a locale).
Then we can synthesise it to a Heyting-valued set.

\begin{construction}
  The set \(\set{f : ­Łĺ×(U, T)}{U Ôłł \h}\) with equality
  \[ \eq f g Ôëö Ôőü\set{V ÔëĄ \e f Ôłž \e g}{f\res V = g\res V} \]
  is the topological Heyting-valued set \(\t T\).
\end{construction}

We now have two synthesisations of \(ÔäŁ\). One is the discrete one \(\c ÔäŁ\), and
the other the topological one \(\t ÔäŁ\). It turns out the latter one is the
object of Dedekind reals.

\begin{example}\label{char:alpo}
  The analytic limited principle of omniscience (\(\alpo*\)) states that the apartness on real numbers is decidable, or \(\for{x : ÔäŁ}{x = 0 ÔłĘ x \apart 0}\).

  Apartness is defined as \(x \apart y Ôëö x < y ÔłĘ x > y\) and \(x < y Ôëö \exist{q:ÔäÜ}{x < q < y}\), where comparisons with rationals are done by containment in the cuts.
  These compute to
  \[ \i{x < y} = (y-x)Ôü╗┬╣(0, Ôł×) \]
  and
  \[ \i{x \apart y} = (y-x)Ôü╗┬╣(ÔäŁ-\{0\})\text. \]

  Then \(\alpo*\) computes to ``for every continuous function \(f : U Ôćĺ ÔäŁ\) we have
  \({ÔŐĄ = \p{f^{-1}(ÔäŁ-\{0\})}^ÔŐą ÔłĘ f^{-1}(ÔäŁ-\{0\})}\).''
  Then \(\alpo*\) is satisfied if and only if for every continuous function
  \(f : ­Łĺ×(p, ÔäŁ)\) \(f^{-1}(ÔäŁ-\{0\})\) is complemented.

  In the topological case, we can rephrase this to say for every continuous map \(f : U Ôćĺ ÔäŁ\) the zero-set \(f^{-1}\{0\}\) is open.
\end{example}
