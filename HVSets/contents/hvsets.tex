\subsection{Preliminaries}

\begin{definition}
  A \emph{Heyting-valued set \(A\)} is a carrier set \(\carr A\) equipped with a ``Heyting-valued relation'' \(\eq[A]{-}{-} : A√óA ‚Üí \h\) satisfying
  \begin{gather}
    \label{hvs:symm}  \eq x y ‚Č§ \eq y x\\
    \label{hvs:trans} \eq x y ‚ąß \eq y z ‚Č§ \eq x z
  \end{gather}

  Define \(\e x ‚ČĒ \eq x x\).
\end{definition}

\begin{definition}
  A \emph{Heyting-valued morphism \(f : A ‚Ü¨ B\)} is a ``Heyting-valued functional relation'', meaning it is a map \(\carr f : \carr A √ó \carr B ‚Üí \h\) satisfying
  \begin{gather}
    \label{hvm:str}    f(x,y) ‚Č§ \e x ‚ąß \e y\\
    \label{hvm:well}   \eq {x'} x ‚ąß f(x, y) ‚ąß \eq y {y'} ‚Č§ f(x', y')\\
    \label{hvm:unique} f(x, y) ‚ąß f(x, y') ‚Č§ \eq y {y'}\\
    \label{hvm:total}  \e x ‚Č§ ‚čĀ_{y ‚ąą B}f(x, y)
  \end{gather}
\end{definition}

\begin{fact}
  The following hold.

  \begin{enumerate}
  \item \(f(x,y)‚ąß\eq x {x'} = f(x',y)‚ąß\eq x {x'}\)
  \item \(\e x = ‚čĀ_{y‚ąąA}\eq x y\)
  \item \(\e x = ‚čĀ_{y‚ąąB}f(x,y)\)
  \item \(f(x, y) = ‚čĀ_{y'‚ąąB}f(x,y')‚ąß\eq y {y'} = ‚čĀ_{x'‚ąąA}f(x',y)‚ąß\eq x {x'}\)
  \end{enumerate}
\end{fact}

\begin{construction}
  The product \(A√óB\) is given by \(\carr A √ó \carr B\) with equality
  \[ \eq{\p{x,y}}{\p{x',y'}} ‚ČĒ \eq x {x'} ‚ąß \eq y {y'}\text. \]

  The projection \(ŌÄ : A√óB ‚Ü¨ A\) is defined by
  \[ \p{\p{x,y}, x'} ‚Ü¶ \eq x {x'} ‚ąß \e y\text. \]
  The other projection is defined similarly
\end{construction}
\begin{proof}
  Symmetry and transitivity of the equality follow clearly.

  Likewise, the projection is clearly strict and well-defined.
  Uniqueness holds via
  \begin{align*}
    ŌÄ(\p{x,y},x') ‚ąß ŌÄ(\p{x,y},x'')
    &= \eq x {x'} ‚ąß \e y ‚ąß \eq x {x''} ‚ąß \e y\\
    &= \eq x {x'} ‚ąß \eq x {x''} ‚ąß \e y\\
    &‚Č§ \eq {x'} {x''} ‚ąß \e y\\
    &‚Č§ \eq {x'} {x''}\text.
  \end{align*}
  Totality holds via
  \begin{align*}
    \e{\p{x,y}} = \e x ‚ąß \e y = ‚čĀ_{x'‚ąąA}\eq x {x'} ‚ąß \e y\text.
  \end{align*}
\end{proof}

\begin{construction}

\begin{fact}
  The empty type \(‚ąÖ\) is given by the empty carrier.

  The unit type \(\mb 1\) is given by \(\{\ast\}\) with \(\e\ast = ‚ä§\). This is

\begin{construction}
  The \emph{exponential object \(B^A\)} is given by
  \[ \set{f : \carr A √ó \carr B ‚Üí \h}{\ref{hvm:str}-\ref{hvm:unique}} \]
  with equality
  \[ \p{f,g} ‚Ü¶ ‚čÄ_{x‚ąąA}‚čĀ_{y‚ąąB}f(x,y) ‚ąß g(x,y)\text. \]
\end{construction}
\begin{internal}
  The equality in the internal language is
  \(\for{x‚ąąA}{\exist{y‚ąąB}{f(x, y) ‚ąß g(x, y)}}\) which can be rephrased as
  \(\for{x‚ąąA}{f(a) = g(a)}\). Thus equality models function extensionality,
  which is know to hold in Grothendieck toposes.
\end{internal}

\begin{construction}
  Given a Heyting-valued set \(A\) and a relation \(R : \carr A √ó \carr A ‚Üí \h\)
  satisfying
  \begin{gather}
    \e x ‚Č§ R x x\\
    R x y ‚Č§ R y x\\
    R x y ‚ąß R y z ‚Č§ R x z
  \end{gather}
  we can form the \emph{quotient \(\quot A R\)} with carrier
  \(\carr A\) and equality \(R\).
\end{construction}
\begin{note}
  Subquotients can be formed similarly, by merely dropping the first condition on the relation.
  If instead \(R x x ‚Č§ \e x\) holds the above construction constructs the subset given by the predicate \(R x x\).
\end{note}

\begin{example}
  We can work out an illustrative example. Consider the principle \(\lem*\),
  which says \(\for{p:ő©}{p ‚ą® ¬¨p}\). Analysing this we get
  \begin{align*}
    \i{\for{p:ő©}{p ‚ą® ¬¨p}}
    &= ‚čÄ_{p‚ąą\h} {\e p ‚áí \i{p ‚ą® ¬¨p}}\\
    &= ‚čÄ_{p‚ąą\h} {\p{p ‚áĒ p} ‚áí p ‚ą® \p{p ‚áí ‚ä•}}\\
    &= ‚čÄ_{p‚ąą\h} {‚ä§ ‚áí p ‚ą® p^‚ä•}\\
    &= ‚čÄ_{p‚ąą\h} {p ‚ą® p^‚ä•}
  \end{align*}

  The principle will hold in \(\h\) precisely when this interpretation equals
  \(‚ä§\). Since it is a conjunction that will hold only when all of its conjuncts
  (\(p ‚ą® p^‚ä•\)) are \(‚ä§\). In other words, \(\lem*\) holds over \(\h\) when
  every element of the Heyting algebra is complemented.
\end{example}
\begin{note}
  The final term in the above computation could be rewritten as \(‚čÄ_{p‚ąą\h\text{ dense}} p\), i.~e. ``the meet of all dense sublocales''. This often simplifies computations, for example, sheaves over the Sierpinski space satisfy \(¬¨¬¨\lem*\) but not \(\lem*\), since \(\lem*\) since the open point is the only non-trivial dense open.
\end{note}

\begin{theorem}
  \(\lem*\) holds over \(\h\) iff \(\h\) is a Boolean algebra.
\end{theorem}

\begin{exercise}
  Show that \(\wlem*\) holds over \(\h\) iff in \(\h\) every pseudo-complement is
  complemented.

  In the case where \(\h\) is a topology this corresponds to extremally
  disconnected spaces (closure of every open set is open).
\end{exercise}
\begin{solution}
  Again we compute
  \begin{align*}
    \i{\for{p:ő©}{¬¨p ‚ą® ¬¨¬¨p}}
    &= ‚čÄ_{p‚ąą\h} {\e p ‚áí \i{¬¨p ‚ą® ¬¨¬¨p}}\\
    &= ‚čÄ_{p‚ąą\h} {p^‚ä• ‚ą® p^{‚ä•‚ä•}}
  \end{align*}
  This then by the same argument means that for every \(p ‚ąą \h\) we have \(p^‚ä•\)
  is complemented.

  In the case of \(\h = ūĚí™X\) saying that an open set is complemented is
  equivalent to saying it is closed. Also, the pseudo-complement in topological
  spaces is the exterior operation (interior of the complement), so this says
  that every exterior of an open set is closed. Then consider what this means
  for complements. The complement of an exterior is the closure, so then this
  says ``the closure of every open set is open'', which is exactly the
  definition of an extremally disconnected space.
\end{solution}

\subsubsection{Intermezzo: Real numbers}

Let \(T\) be some topological space. Then we can synthesise it to a
Heyting-valued set.

\begin{construction}
  The set \(\set{f : ūĚíě(U, T)}{U ‚ąą \h}\) with equality
  \[ \eq f g ‚ČĒ ‚čĀ\set{V ‚Č§ \e f ‚ąß \e g}{f\res V = g\res V} \]
  is the topological Heyting-valued set \(\t X\).
\end{construction}

We now have two synthesisations of \(‚ĄĚ\). One is the discrete one \(\c ‚ĄĚ\), and
the other the topological one \(\t ‚ĄĚ\). It turns out the latter one is the
object of Dedekind reals.

\begin{example}
  The analytic limited principle of omniscience (\(\alpo*\)) states that the apartness on
  real numbers is decidable, or \(\for{x : ‚ĄĚ}{x = 0 ‚ą® x \apart 0}\).

  Apartness is defined as \(x \apart y ‚ČĒ x < y ‚ą® x > y\) and
  \(x < y ‚ČĒ \exist{q:‚Ąö}{x < q < y}\), where comparisons with rationals are done
  by containment in the cuts. These compute to
  \[ \i{x < y} = (y-x)‚ĀĽ¬Ļ(0, ‚ąě) \]
  and
  \[ \i{x \apart y} = (y-x)‚ĀĽ¬Ļ(‚ĄĚ-\{0\})\text. \]

  Then \(\alpo*\) computes to ``for every continuous function \(f : U ‚Üí ‚ĄĚ\) we
  have \({‚ä§ = \p{f^{-1}(‚ĄĚ-\{0\})}^‚ä• ‚ą® f^{-1}(‚ĄĚ-\{0\})}\).''
  Then \(\alpo*\) is satisfied if and only if for every continuous function
  \(f : ūĚíě(p, ‚ĄĚ)\) \(f^{-1}(‚ĄĚ-\{0\})\) is complemented.

  In the topological case, we can rephrase this to say for every continuous map
  \(f : U ‚Üí ‚ĄĚ\) the zero-set \(f^{-1}\{0\}\) is open.
\end{example}

A similar computation can be carried out for the Cantor space \(\t{2^‚Ąē}\). It
turns out this space is just the object power-object \(2^‚Ąē\).

\begin{exercise}
  Let \(\h\) be a second-countable topological space (or a localic equivalent).
  Show that \(\lpo*\) holds over \(\h\) iff \(\h\) is locally connected.

  Note: This will need some topological computation, but a useful lemma is that a space is locally connected iff union of relatively clopen subsets of \(U\) is again relatively clopen in \(U\). You may take this as given (it is not part of the exercise per se, but it is what needs to be done in practice to obtain useful characterisations).
\end{exercise}
\begin{solution}
  First compute what \(\lpo*\) means.
  \begin{align*}
    \i{\for{őĪ:2^‚Ąē}{őĪ \apart 0 ‚ą® őĪ = 0}}
    &= ‚čÄ_{őĪ‚ąą2^‚Ąē_{\h}} {\e őĪ ‚áí \i{őĪ \apart 0 ‚ą® őĪ = 0}}\\
    &= ‚čÄ_{őĪ‚ąą2^‚Ąē_{\h}} {\e őĪ ‚áí \i{őĪ \apart 0} ‚ą® \i{őĪ = 0}}
  \end{align*}
  We may now reinterpret each \(őĪ ‚ąą \t{2^‚Ąē}\) as a sequence of relatively clopen sets in \(\e őĪ\), namely the sequence \(k ‚Ü¶ \i{őĪ‚āĖ = 1}\). Thus, \(\lpo*\) holds iff for all \(p ‚ąą \h\) and all sequences of relatively clopen sets in \(p\) we have \(p ‚Č§ ‚čĀ‚āôP‚āô ‚ą® ‚čÄ‚āô P‚āôŠ∂ú\).
  In other words ``every sequence of relatively clopen sets has a relatively clopen union''.

  Applying second-countability and the above allows us to pass from a countable sequence to an arbitrary family of relatively clopen sets, which completes the proof.

  Let \(U‚āô ‚ČĒ \i{őĪ n = 1}\). This is a clopen set with complement \(\i{őĪ n = 0}\).
  This is a sequence of relatively clopen sets in \(\e őĪ\).
  Conversely, if \(U‚āô\) is a sequence of relatively clopen sets in \(U ‚ąą \h\) we can form the map \(őĪ : ūĚíě(U, 2^‚Ąē)\) by \(t ‚Ü¶ őĽ n : ‚Ąē.\; t ‚ąą U‚āô\).
  For this map we have \(\i{őĪ n = 1} = U‚āô\).

  Thus \(\lpo*\) holds iff for every sequence of relatively clopen sets their intersection is open (and thus clopen) or equivalently, if their union is closed.

  To make the step from closure under countable unions to arbitrary unions we use second-countability of the space.
  Let \(\{B‚āô\}\) be a countable basis for the space and \(\{VŠĶĘ\}_{i‚ąąI}\) a family of relatively clopen subsets of \(U\).
  Define
  \[
    W‚āô ‚ČĒ
    \begin{cases}
      VŠĶĘ \quad; \text{for any \(i\) such that \(B‚āô ‚äÜ VŠĶĘ\)}\\
      ‚ąÖ  \quad; \text{if no such \(i\) exists.}
    \end{cases}
  \]
  Now I claim that \(‚čÉ_{n‚ąą‚Ąē}W‚āô = ‚čÉ_{i‚ąąI}VŠĶĘ\).
  Take \(x ‚ąą VŠĶĘ\). Then since there is some basic \(B‚āô ‚äÜ VŠĶĘ\) containing \(x\), we know that \(W‚āô = V‚Īľ\) for some \(j‚ąąI\), such that \(x ‚ąą B‚āô ‚äÜ V‚Īľ\), therefore \(x ‚ąą W‚āô\).
  Thus the lattice of clopen sets is complete.

  To show that the space is locally connected recall that connected components are intersections of all clopen sets containing them. Then by the above the intersection of clopen sets is again clopen, therefore the connected components of the space are open, which completes the proof.
\end{solution}

\end{note}
