\subsection{Preliminaries}

\begin{definition}
  A \emph{Heyting-valued set \(A\)} is a carrier set \(\carr A\) equipped with a ``Heyting-valued relation'' \(\eq[A]{-}{-} : A├ŚA Ôćĺ \h\) satisfying
  \begin{gather}
    \label{hvs:symm}  \eq x y ÔëĄ \eq y x\\
    \label{hvs:trans} \eq x y Ôłž \eq y z ÔëĄ \eq x z
  \end{gather}

  Define \(\e x Ôëö \eq x x\).
\end{definition}

\begin{definition}
  A \emph{Heyting-valued morphism \(f : A ÔćČ B\)} is a ``Heyting-valued functional relation'', meaning it is a map \(\carr f : \carr A ├Ś \carr B Ôćĺ \h\) satisfying
  \begin{gather}
    \label{hvm:str}    f(x,y) ÔëĄ \e x Ôłž \e y\\
    \label{hvm:well}   \eq {x'} x Ôłž f(x, y) Ôłž \eq y {y'} ÔëĄ f(x', y')\\
    \label{hvm:unique} f(x, y) Ôłž f(x, y') ÔëĄ \eq y {y'}\\
    \label{hvm:total}  \e x ÔëĄ Ôőü_{y Ôłł B}f(x, y)
  \end{gather}
\end{definition}

\begin{fact}
  The following hold.

  \begin{enumerate}
  \item \(f(x,y)Ôłž\eq x {x'} = f(x',y)Ôłž\eq x {x'}\)
  \item \(\e x = Ôőü_{yÔłłA}\eq x y\)
  \item \(\e x = Ôőü_{yÔłłB}f(x,y)\)
  \item \(f(x, y) = Ôőü_{y'ÔłłB}f(x,y')Ôłž\eq y {y'} = Ôőü_{x'ÔłłA}f(x',y)Ôłž\eq x {x'}\)
  \end{enumerate}
\end{fact}

\begin{construction}
  The product \(A├ŚB\) is given by \(\carr A ├Ś \carr B\) with equality
  \[ \eq{\p{x,y}}{\p{x',y'}} Ôëö \eq x {x'} Ôłž \eq y {y'}\text. \]

  The projection \(¤Ç : A├ŚB ÔćČ A\) is defined by
  \[ \p{\p{x,y}, x'} ÔćŽ \eq x {x'} Ôłž \e y\text. \]
  The other projection is defined similarly
\end{construction}
\begin{proof}
  Symmetry and transitivity of the equality follow clearly.

  Likewise, the projection is clearly strict and well-defined.
  Uniqueness holds via
  \begin{align*}
    ¤Ç(\p{x,y},x') Ôłž ¤Ç(\p{x,y},x'')
    &= \eq x {x'} Ôłž \e y Ôłž \eq x {x''} Ôłž \e y\\
    &= \eq x {x'} Ôłž \eq x {x''} Ôłž \e y\\
    &ÔëĄ \eq {x'} {x''} Ôłž \e y\\
    &ÔëĄ \eq {x'} {x''}\text.
  \end{align*}
  Totality holds via
  \begin{align*}
    \e{\p{x,y}} = \e x Ôłž \e y = Ôőü_{x'ÔłłA}\eq x {x'} Ôłž \e y\text.
  \end{align*}
\end{proof}

\begin{construction}
  Given a Heyting-valued set \(A\) and a relation \(R : \carr A ├Ś \carr A Ôćĺ \h\)
  satisfying
  \begin{gather}
    \e x ÔëĄ R x x\\
    R x y ÔëĄ R y x\\
    R x y Ôłž R y z ÔëĄ R x z
  \end{gather}
  we can form the \emph{quotient \(\quot A R\)} with carrier
  \(\carr A\) and equality \(R\).
\end{construction}
Subquotients can be formed similarly, by merely dropping the first condition on
the relation.

\begin{example}
  We can work out an illustrative example. Consider the principle \(\lem*\),
  which says \(\for{p:╬ę}{p ÔłĘ ┬Čp}\). Analysing this we get
  \begin{align*}
    \i{\for{p:╬ę}{p ÔłĘ ┬Čp}}
    &= ÔőÇ_{pÔłł\h} {\e p Ôçĺ \i{p ÔłĘ ┬Čp}}\\
    &= ÔőÇ_{pÔłł\h} {\p{p Ôçö p} Ôçĺ p ÔłĘ \p{p Ôçĺ ÔŐą}}\\
    &= ÔőÇ_{pÔłł\h} {ÔŐĄ Ôçĺ p ÔłĘ p^ÔŐą}\\
    &= ÔőÇ_{pÔłł\h} {p ÔłĘ p^ÔŐą}
  \end{align*}

  The principle will hold in \(\h\) precisely when this interpretation equals
  \(ÔŐĄ\). Since it is a conjunction that will hold only when all of its conjuncts
  (\(p ÔłĘ p^ÔŐą\)) are \(ÔŐĄ\). In other words, \(\lem*\) holds over \(\h\) when
  every element of the Heyting algebra is complemented.
\end{example}

\begin{theorem}
  \(\lem*\) holds over \(\h\) iff \(\h\) is a Boolean algebra.
\end{theorem}

We can show a weaker version the same way.

\begin{exercise}
  Show that \(\wlem*\) holds over \(\h\) iff in \(\h\) every pseudo-complement is
  complemented.

  In the case where \(\h\) is a topology this corresponds to extremally
  disconnected spaces (closure of every open set is open).
\end{exercise}
\begin{solution}
  Again we compute
  \begin{align*}
    \i{\for{p:╬ę}{┬Čp ÔłĘ ┬Č┬Čp}}
    &= ÔőÇ_{pÔłł\h} {\e p Ôçĺ \i{┬Čp ÔłĘ ┬Č┬Čp}}\\
    &= ÔőÇ_{pÔłł\h} {p^ÔŐą ÔłĘ p^ÔŐą^ÔŐą}\\
  \end{align*}

  This then by the same argument means that for every \(p Ôłł \h\) we have \(p^ÔŐą\)
  is complemented.

  In the case of \(\h = ­Łĺ¬X\) saying that an open set is complemented is
  equivalent to saying it is closed. Also, the pseudo-complement in topological
  spaces is the exterior operation (interior of the complement), so this says
  that every exterior of an open set is closed. Then consider what this means
  for complements. The complement of an exterior is the closure, so then this
  says ``the closure of every open set is open'', which is exactly the
  definition of an extremally disconnected space.
\end{solution}

\subsubsection{Intermezzo: Real numbers}

Let \(T\) be some topological space. Then we can synthesise it to a
Heyting-valued set.

\begin{construction}
  The set \(\set{f : ­Łĺ×(U, T)}{U Ôłł \h}\) with equality
  \[ \eq f g Ôëö Ôőü\set{V ÔëĄ \e f Ôłž \e g}{f\res V = g\res V} \]
  is the topological Heyting-valued set \(X_{\h}\).
\end{construction}

\begin{example}
  The analytic limited principle of omniscience (\(\alpo*\)) states that the apartness on

\begin{exercise}
  Let \(\h\) be a second-countable topological space (or a localic equivalent).
  Show that \(\lpo*\) holds over \(\h\) iff \(\h\) is locally connected.

  Note: This will need some topological computation, but a useful lemma is that
  a space is locally connected iff intersection of relatively clopen subsets of
  \(U\) is again relatively clopen in \(U\). You may take this as given (it is
  not part of the exercise per se, but it is what needs to be done in practice
  to obtain useful characterisations).
\end{exercise}
\begin{solution}
  First compute what \(\lpo*\) means.
  \begin{align*}
    \i{\for{╬▒:2^ÔäĽ}{╬▒ \apart 0 ÔłĘ ╬▒ = 0}}
    &= ÔőÇ_{╬▒Ôłł2^ÔäĽ_{\h}} {\e ╬▒ Ôçĺ \i{╬▒ \apart 0 ÔłĘ ╬▒ = 0}}\\
    &= ÔőÇ_{╬▒Ôłł2^ÔäĽ_{\h}} {\e ╬▒ Ôçĺ \p{ÔőÇ_{nÔłłÔäĽ} \i{╬▒ n = 1}} ÔłĘ \p{Ôőü_{nÔłłÔäĽ} \i{╬▒ n = 0}}}
  \end{align*}

  Let \(UÔéÖ Ôëö \i{╬▒ n = 1}\). This is a clopen set with complement \(\i{╬▒ n = 0}\).
  This is a sequence of relatively clopen sets in \(\e ╬▒\).
  Conversely, if \(UÔéÖ\) is a sequence of relatively clopen sets in \(U Ôłł \h\) we can form the map \(╬▒ : ­Łĺ×(U, 2^ÔäĽ)\) by \(t ÔćŽ ╬╗ n : ÔäĽ.\; t Ôłł UÔéÖ\).
  For this map we have \(\i{╬▒ n = 1} = UÔéÖ\).

  Thus \(\lpo*\) holds iff for every sequence of relatively clopen sets their intersection is open (and thus clopen).

  To make the step from closure under countable intersections to arbitrary intersections we use second-countability of the space.

\end{solution}
